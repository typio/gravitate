# Gravy Galaxy

> 3D Gravity (N-body) Simulation

It's gravy like the soup

## Usage

The project is compiled by wasm-pack from rust to an npm package.

```bash
$ pnpm i gravy-galaxy
```

Currently it builds for WebGL but that will be changed to webGPU when it's released.
